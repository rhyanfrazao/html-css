Principais Formatações

Negrito / Destaque

Nesta frase, temos um termo em negrito usando a tag B (não semântica).

Nesta frase, temos um termo em destaque usando a tag STRONG (semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag I (não semântica).

Nesta frase, temos um termo em ênfase usando a tag EM (semântica).

Texto Marcado

Podemos criar também um texto marcado usando a tag MARK.

E podemos editar este termo usando o style pontual.

Textos Grandes e Pequenos

Estamos criando um texto grande e um texto pequeno com as tags big e small. Apenas a small ainda funciona por conta da semântica de letras pequenas em contratos.

Texto Deletado

Podemos marcar um texto como excluído para mostrar que ele estava no texto original, mas não deve ser considerado.

Texto Sublinhado e Texto Inserido

Nesta frase, temos um termo sublinhado utilizando a tag B (não semântica).

Nesta frase, temos um termos inseridos utilizando a tag INS (semântica).

Texto Sobescrito

Para inserir termos como x2 + 3x - 1. Utilizamos a tag SUP.

Texto Subscrito

Para inserir termos como H2O. utilizamos a tag SUB.
