Principais Formatações

Negrito/Destaque

Nessa frase temos um termo em negrito usando a tag B. Porém essa não é uma tag semântica.

Nessa frase tenho um termo em destaque usando a tag STRONG. Essa sim é uma tag semântica.

Itálico/Ênfase

Nessa frase temos um termo em itálico usando a tag I. Porém essa não é uma tag semântica.

Nessa frase temos um termo em ênfase usando a tag EM. Essa sim é uma tag semântica.

Texto Marcado

Podemos criar também um texto marcado com a tag MARK.

Texto Grande e Texto Pequeno

Estamos criando um texto grande e um texto pequeno. A tag BIG se tornou obsoleta por não dar sentido ao texto, a SMALL não, pois ainda existem as "letras pequenas" em contratos.

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, porém não mais considerado.

Texto Inserido

Podemos marcar um texto como inserido para indicar que ele foi adicionado posteriormente à elaboração do documento original.

Texto Sobrescrito

Para inserir coisas do tipo: 22 = 4

Texto Subscrito

Para inserir coisas do tipo: H2O
